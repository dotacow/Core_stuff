-----------**Commands**

(creating user) sudo adduser <username>
(adding user to group) sudo usermod -aG <groupname> <username>
(group members) getent group <groupname>

(path of config for sshd) etc/ssh/shhd_config
(hostname and hosts path) etc/hostname then etc/hosts

(checking if installed) dpkg -l |grep <thing>

(ssh) sudo systemctl status ssh
(ufw) sudo ufw status
(pswd rules) sudo chage -l username
